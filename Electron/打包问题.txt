windows下使用nsis打包时，镜像设置为淘宝npm;
npm config set ELECTRON_MIRROR http://npm.taobao.org/mirrors/electron/
或者编辑用户目录下.npmrc文件，后新增ELECTRON_MIRROR=http://npm.taobao.org/mirrors/electron/

electron-vue process is not defined报错方法，在
.electron-vue里面的webpack.renderer.config.js和webpack.web.config.js中新增
new HtmlWebpackPlugin({
      filename: 'index.html',
      template: path.resolve(__dirname, '../src/index.ejs'),
      templateParameters(compilation, assets, options) {
        return {
          compilation: compilation,
          webpack: compilation.getStats().toJson(),
          webpackConfig: compilation.options,
          htmlWebpackPlugin: {
            files: assets,
            options: options
          },
          process,
        };
      },
      minify: {
        collapseWhitespace: true,
        removeAttributeQuotes: true,
        removeComments: true
      },
      nodeModules: false
    }),
